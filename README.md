# Flipping-Card

Stack: HTML, CSS, JS  
Status: działający  
Demo: https://mikel538.github.io/Flipping-card/

Interaktywny komponent obracającej się karty wykonany w HTML, CSS i JavaScript.

Jest to jedno z pierwszych użyć JS bez pomocy instrukcji typu film na Youtube krok po kroku.

Cel projektu:

- praktyczne użycie JavaScript do obsługi interakcji
- implementacja animacji i transformacji CSS
- łączenie logiki JS z warstwą UI
